# 1collective

Technology Stack for Blue Collar Workers — a platform that brings modern digital tools to tradespeople, contractors, and field workers.

## Project Overview

A React + Vite (TypeScript) single-page application. No backend — frontend only.

## Stack

- React 19 + TypeScript
- Vite 8 (dev server on port 5000, host 0.0.0.0)
- react-router-dom for routing
- lucide-react for icons

## Running the App

```bash
npm run dev
```

## User Preferences

- No emojis in the UI
- Clean, professional look aimed at blue collar / trade workers
